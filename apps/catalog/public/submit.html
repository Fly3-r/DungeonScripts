How Submission Works

There is no public admin page. Reviews happen privately on the host with local CLI tooling.

What To Include

Provide all four script fields, a stable package ID, version, an AI Dungeon author profile URL, and any post-install notes in the description.

Privacy

Your Discord username is stored privately with the submission so the host can contact you. It is never exposed in the public catalog APIs.

Description Format

Descriptions may be long and can include Markdown links such as [Guide](https://example.com).
